| /metrics endpoint exposes internal counters publicly | Medium | Bound to 127.0.0.1 via reverse proxy; Prometheus scrapes internally on Docker network; /metrics not exposed to internet |
| Grafana admin password weak default | Medium | Default is 'admin' only if GRAFANA_ADMIN_PASSWORD not set; bound to localhost:3000; user must set strong password |
| RESEND_API_KEY in environment variable | Low | Standard practice for SMTP secrets; env var not logged; docker-compose reads from .env file |
| Prometheus data retention (30d) disk usage | Low | 30d retention configured; prometheus-data volume is separate; can be tuned |
| Alert fatigue from false positives | Low | 5-minute `for` duration prevents transient failures from triggering alerts; noDataState=Alerting catches complete probe failures |
</threat_model>
